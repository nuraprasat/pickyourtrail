# pickyourtrail

Run each class file as simple java application, If you are running in eclipse give the input in the console.
or
javac classname.java - give each class name
java classname inputs

Example 1:
input : 
3 - size
1
2
2

output:
6

Example 2:
input :
3 - size
push 3
push 2
pop

output:
3
2
3

Example 3:
input:
4 - size
1
1
2
2
1 - difference (k value)

output:
1
